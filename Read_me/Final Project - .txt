Final Project - 

	User based recommender system to find places of interest in the country of Canada

Data- 
	Ticket Master API,  Yelp fussion API, FourSquare API

Purpose-
	 To deploy a model to return places in a city in Canada with events and business around in a small radius , with ratings (preferable the top buisinesses ). The idea behind is to create an application to assist a user in creating an itenirary  based on the users history of ratings and also suggest some other things of interest based on other users.

Main aim - 
	To be able to retrieve data fromAPI's and parse json files. Distinguish between relevant important data from all the information available. 
	Build a recommender system 
	Work with deployement of an application using streamlit.io

Process- 
	use endpoint /discovery/v2/events for ticket master with parameters of  country code- Canada, segment - mucis, arts and theatre, sports
	create a data frame with only lat, lon and venue to get business from yelp fussion api using end point /v3_business_search with the parameters of latitude,longitude and radius
	retrieve data of ratings, review count, price,hours of operation 
	merge this detais with other information retrieved from ticket master API 
	deploy it using streamlit.io
	preferable user can add city of choice
